# Memcached emulator

This proyect intend is to emulate memcached, using a personal implementation following the memcached protocol

## Necessary resources

In order to run this project you must have [NodeJs](https://nodejs.org/es/download/) installed.

## Commands suported

Retrieval commands:
* get
* gets

Storage commands:
* set
* add
* replace
* append
* prepend
* cas
